# STUDENTS-SCORESHEET
This project is a Java-based Student Report Generator that collects school details, student names, marks for various subjects, computes totals, averages, and assigns ranks. It then prints a well-formatted report.

FEATURES

Input school information (School Name, Teacher Name, Grade, Year)

Input student names and marks for:

English

Math

History

Geography

Science

Programming

Automatic computation of:

Total marks per student

Rank assignment (A, B, C, D, F)

Subject totals

Subject averages


RANK CALCULATION CRITERIA
Total Marks	Rank
540 and above	A
480 and above	B
420 and above	C
360 and above	D
Below 360 Grade F

How to Run the Program

Install Java JDK (17 or later recommended)

Save the file as Main.java

Compile using:

javac Main.java

Run the program: java main
